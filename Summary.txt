Git merupakan perangkat lunak untuk memanajemen kode perangkat lunak
Github merupakan layanan hos web. Fungsi utama github yaitu sebagai media penyimpanan untuk berkolaborasi bersama sama dalam suatu projek.
Tahap awal dalam versioning control dilakukan yaitu menginstal softwere git 
Kemudian membuat akun gitgub
Lalu buat folder pada dokumen di komputer
Kemudian buat file 
Setelah file terbuat buka git untuk menghubungkan git dengan git hub.

1.	git init
2.	git add README.md
3.	git commit -m "first commit"
4.	git branch -M master
5.	git remote add origin link github 
6.	git push -u origin master
